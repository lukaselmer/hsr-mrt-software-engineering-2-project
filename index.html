MRT - Mobile Reporting Tool

MRT steht für Mobile Reporting Tool und bietet Firmen mit Aussendienstmitarbeitern eine einfache Möglichkeit, Arbeitsstunden zu rapportieren.

Unter verzeichnis_und_uebersicht.docx ist eine Übersicht über die Projektstruktur und im Projektplan eine Einführung ins Thema MRT zu finden. Die Teammitglieder sind ebenfalls im Projektplan aufgelistet.

Im Ordner doc sind alle Dokumente enthalten. Um die Informationen nicht redundant darzustellen, wird auf die jeweiligen Dokumente verwiesen.

Für eine interaktive Einführung und für die wichtigsten Eckdaten steht eine Präsentation zur Verfügung.
